## 安装

- Windows 上需要 64 位的 rime.dll（仓库里已经有一个编译好的）。
- Linux 上不可用，待开发。

## 使用

~~不会真有人想用这玩意吧？~~

将 rime 用户目录的 build 文件夹复制到机器人根目录。
